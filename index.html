컬러 페이지 색출기

PDF / HWPX / HWP 보고서 파일에서 컬러 페이지를 찾아냅니다. (브라우저에서만 동작, 서버 전송 없음)
⚠ 정확한 페이지별 결과가 필요하면 → 한글에서 PDF로 저장한 뒤 업로드하세요
  • PDF: 각 페이지를 실제 이미지처럼 렌더링한 뒤 픽셀 색상을 분석합니다 → 페이지/쪽번호별 정확한 판정 가능.
  • HWPX(한글 신형식): 워드 문서처럼 글이 흐르는 구조라 XML만으로는 "몇 페이지째"인지 알 수 없습니다. 포함된 이미지·색상 정보는 보여주지만 페이지 단위 결과가 아닌 문서 전체 참고용입니다. → 정확한 페이지가 필요하면 한글에서 [파일 → 다른 이름으로 저장 → 파일 형식: PDF]로 저장한 뒤 그 PDF를 업로드해 주세요.
  • HWP(구버전 바이너리): 브라우저에서 안전하게 시각적으로 분석할 방법이 없어 미지원입니다. 위와 동일하게 PDF로 변환 후 업로드해 주세요.
※ 이 변환은 한글 프로그램에서 직접 한 번만 눌러주시면 되고, 자동화는 지원하지 않습니다 (브라우저 보안 정책상 불가능).
※ 인쇄요청서 PDF의 파란 테두리 칸은 자유롭게 직접 입력·수정할 수 있습니다.
